Техническое задание
Название проекта: I cant breathe

Жанр: 2D top-down survival

1. Общая идея игры
Игрок управляет персонажем, оказавшимся на изолированной станции без кислорода. Основная задача — добраться до выхода, управляя запасом кислорода, избегая врагов и собирая баллоны с O₂.
2. Основные механики
• Передвижение игрока с помощью клавиш WASD
• Ограниченный запас кислорода (таймер)
• Подбор кислородных баллонов
• Столкновения со стенами и врагами

3. Игровые объекты
Игрок:
• Имеет запас кислорода
• Может перемещаться по уровню

Враги:
• Патрулируют уровень
• Наносят урон при столкновении

Предметы:
• Кислородные баллоны
• Двери и выход

4. Уровни
Игра содержит несколько уровней.
С каждым уровнем увеличивается сложность:
• меньше кислорода
• больше врагов
• усложнённая карта

5. Интерфейс
• Стартовое меню с кнопкой начала игры
• Игровой HUD (индикатор кислорода)
• Финальный экран с результатами

6. Звук и анимация
• Звук дыхания
• Предупреждение о низком уровне кислорода
• Анимация движения игрока

7. Технические требования
• Язык программирования: Python
• Библиотека: Arcade
• Использование физического движка Pymunk

• Система частиц
• Камера
• Хранение данных в CSV

8. Работа с репозиторием
Проект ведётся через сетевой репозиторий.
Каждая функциональная часть реализуется в отдельной ветке.
